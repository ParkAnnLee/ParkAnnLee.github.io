Permutations

Input

Array A of distinct numbers

Output

Permutations of A

Example

Input: A = [1,2]
Output: [[1, 2], [2, 1]]

Note

If A has n elements, the number of permutations is n! The number of permutations is way larger than the number of combinations.

Idea of implementing permutations

Suppose we have a list [1, 2, 3]
Permutations without 3 will be [[1, 2], [2, 1]]
Solution with 3 is [[3, 1, 2], [1, 3, 2], [1, 2, 3], [3, 2, 1], [2, 3, 1], [2, 1, 3]]

Puesdocode

Multiplications: n

Goal: Less multiplications

\(2^{1000} = 2^{500} \times 2^{500}\)

  1. compute \(2^{500}\)
  2. square it

\(2^{500} = 2^{250} \times 2^{250}\)

  1. compute \(2^{250}\)
  2. square it

Total: 252 multiplications

\(2^{125} = 2^{62.5} \times 2^{62.5} = 2^{62} \times 2^{62} \times 2\)
