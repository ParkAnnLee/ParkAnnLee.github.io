Complexity Analysis

\(T(n) = 2\times T(\frac{n}{2}) + \Theta(n)\)
\(T(n) = \log_2{n}\times\Theta(n) = O(n\log{n})\)

What if
\(T(n) = 2\times T(\frac{n}{2}) + n^2\)
\(T(n) = n^2\times\frac{1-\frac{1}{2}^{\log_2{n}}}{1-\frac{1}{2}} = O(n^2)\)
