Subset Sum

Input

  • array A[1..n] of positive integers,
  • positive integer s

Output

  • True if s can be written as the sum of a subset of A
  • False otherwise

Notation

SS[i, j] =

  • True: if j can be written as a sum of a subset of A[1..i]
  • False: otherwise

SS[i, j] = SS[i-1, j] or SS[i-1, j-A[i]]
